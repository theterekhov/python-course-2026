# Задание 6: REPL для работы с почтовыми индексами США

Консольное приложение для поиска информации о почтовых индексах США.
Использует набор данных из файла `zip_codes_states.csv` и модуль `zip_util.py`.

## Запуск

```bash
python3 solution.py
```

## Команды

- **loc** — найти местоположение по почтовому индексу
- **zip** — найти почтовые индексы по городу и штату
- **dist** — вычислить расстояние между двумя почтовыми индексами
- **end** — завершить работу

Команды нечувствительны к регистру.

## Пример

```
Command ('loc', 'zip', 'dist', 'end') => loc
loc
Enter a ZIP Code to lookup => 12180
12180
ZIP Code 12180 is in Troy, NY, Rensselaer county,
coordinates: (042∘ 40'25.32"N,073∘ 36'31.65"W)
```
