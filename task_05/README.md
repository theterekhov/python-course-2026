# Задание 5: Игра Жизнь

Консольная реализация игры "Жизнь" Джона Конвея (Conway's Game of Life).

## Запуск

```bash
python3 solution.py
```

Программа последовательно запрашивает:
1. Имя входного файла с конфигурацией поля
2. Количество шагов моделирования
3. Базовый цвет в формате RRGGBB (например, FF0000 для красного)
4. Префикс для выходных файлов

## Формат входного файла

Первая строка содержит два числа: количество строк и столбцов.
Далее идут строки поля, где `.` — мёртвая клетка, `O` — живая клетка.

Пример (`input.txt`):
```
10 30
..............................
..............................
..............................
..............................
..........OOOO........OOOO....
.........O....O......O....O...
.........O....O......O....O...
..........OOOO........OOOO....
..............................
..............................
```

## Выходные данные

- `{префикс}.txt` — текстовая запись всех поколений
- `{префикс}_0.png`, `{префикс}_1.png`, ... — изображения каждого поколения

Цвет живых клеток зависит от их возраста: чем старше клетка, тем насыщеннее цвет.
